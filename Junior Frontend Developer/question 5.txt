Question #5 - Junior Level
1. What is semantic HTML?
2. Why you would like to use semantic tag?

Answer
1. Semantic คือ คุณสมบัติของ HTML5 ที่ช่วยในการจัดหมวดหมู่ของเนื้อหาต่างบนเว็บเพจ ซึ่งจะใช้ Semantic Elements ซึ่งเป็นแท็กที่จะช่วยในการจัดหมวดหมู่ ซึ่งแต่ละแท็ก ของ Semantic Elements ซึ่งผู้พัฒนาเว็บ จะต้องใช้คำสั่งแท็กให้ถูกต้องและตรงความหมายมากที่สุด

ตัวอย่างแท็ก Semantic Elements
แท็ก <HEADER>
หมายถึง ส่วนที่เป็นส่วนหัวของเว็บเพจ
แท็ก <NAV>
หมายถึง ส่วนที่เป็นเมนูของเว็บเพจ หรือลิงค์ไปยังหน้าเว็บเพจต่างๆ
แท็ก <ARTICLE>
หมายถึง เนื้อหาของเว็บ
แท็ก <SECTION>
หมายถึง กลุ่มของหัวข้อย่อย
แท็ก <ASIDE>
หมายถึง เนื้อหาส่วนเสริม ที่แยกจากเนื้อหาหลัก
แท็ก <FOOTER>
หมายถึง ส่วนท้ายของหน้าเว็บเพจ เช่น copyright เป็นต้น

2. ทำให้ Element อ่านง่ายมากขึ้น และช่วยให้ เราทำ SEO ได้มีประสิทธิภาพมากยิ่งขึ้นซึ่งจะส่งผลดีในแง่ของ search engine อย่างมาก
